- [less modules](http://lesscss.org/)
- [prettier](https://prettier.io/)
- [debug](https://www.npmjs.com/package/debug)  
  ...

## Installation

- clone repos:

```shell script
$ git clone git@github.com:willybrauner/webpack-base.git
```

- move to folder:

```shell script
$ cd webpack-base
```

- run install script:

```shell script
$ ./install.sh
```
